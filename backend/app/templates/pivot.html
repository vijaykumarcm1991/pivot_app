{# Load order matters: pivot grid wrapper, pivot export, then the controller,
   then the drill-down modules, then the email modules. The drill-down /
   email modules rely on window.PivotGrid + window.PivotExport + the pivot
   controller's appState being available.

   The version query string forces every browser to re-fetch the JS files
   on page load. Without it, AG Grid's "values are showing empty" fix
   (destroy+recreate on every render) + the soft-delete row-key fix
   would not propagate to users who had the page open before the fixes
   were deployed. Bump the version whenever a JS file in this list changes. #}
